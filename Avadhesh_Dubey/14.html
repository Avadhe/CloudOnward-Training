Responsive Images and Media

Responsive images and media ensure that visual content scales properly across different devices and screen sizes.

Responsive Images:

To make images responsive, use the max-width property to ensure images adapt to the width of their container.

img {
    max-width: 100%;
    height: auto;
}

Responsive Media:

For videos and other media, set the width to 100% to ensure they scale with their container.

.responsive-video {
    width: 100%;
    height: auto;
}
