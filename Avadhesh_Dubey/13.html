Advanced Layout Techniques (Grid, Responsive Design)

Advanced layout techniques like CSS Grid and responsive design enhance the flexibility and adaptability of web layouts.

CSS Grid:

CSS Grid Layout is a powerful layout system for creating complex and responsive grid-based designs. It allows for precise control over rows, columns, and spacing.

.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
}

.grid-item {
    background-color: #f4f4f4;
    padding: 20px;
    text-align: center;
}

Responsive Design:

Responsive design ensures that a webpage looks good on all devices by using flexible grids, images, and media queries.

@media screen and (max-width: 768px) {
    .container {
        width: 100%;
        padding: 10px;
    }
}
