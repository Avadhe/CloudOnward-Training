Structure of an HTML Document

The structure of an HTML document consists of several key elements that define how the content is organized and displayed in the browser. Understanding this structure is crucial for writing valid and effective HTML.

Key Sections of an HTML Document:

  • <html> - The root element that encloses the entire HTML document.
  • <head> - Contains meta-information about the document, such as the title and character encoding.
  • <title> - Sets the title of the document, which appears in the browser's title bar or tab.
  • <meta> - Provides metadata about the document, such as the character set and viewport settings.
  • <link> - Links external resources, like stylesheets, to the document.
  • <body> - Contains the main content of the document, including text, images, and other elements.
  • <header>, <main>, <footer> - Semantic elements that help structure the content logically.
