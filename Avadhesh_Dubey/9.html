What is CSS and How it Works

Cascading Style Sheets (CSS) is a stylesheet language that describes the presentation of a document written in HTML. It allows you to apply styles to web documents and control the layout and appearance of elements.

How CSS Works:

  • CSS Rules: A CSS rule consists of a selector and a declaration block. The selector targets HTML elements, and the declaration block contains property-value pairs (e.g., color: red;).
  • Applying Styles: CSS can be applied in three ways: inline styles (using the style attribute), internal styles (within the <style> tag in the <head>), and external styles (using a separate CSS file linked via the <link> tag).
  • Cascade and Specificity: CSS rules follow a cascade, meaning styles are applied in a specific order. Specificity determines which rule is applied if multiple rules target the same element.
