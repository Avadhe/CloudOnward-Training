CSS Animations and Transitions

CSS animations and transitions add dynamic effects to web elements, enhancing user experience with smooth changes in styles.

Transitions:

CSS transitions allow you to change property values smoothly over a given duration.

.animated-element {
    width: 100px;
    height: 100px;
    background-color: #3498db;
    transition: background-color 0.5s ease;
}

.animated-element:hover {
    background-color: #e74c3c;
}

Animations:

CSS animations enable keyframe-based animation, allowing for more complex animations beyond simple transitions.

@keyframes slideIn {
    from {
        transform: translateX(-100%);
        opacity: 0;
    }
    to {
        transform: translateX(0);
        opacity: 1;
    }
}

.slide-in {
    animation: slideIn 1s ease-out;
}
