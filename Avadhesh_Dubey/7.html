Practical Exercise: Create a Simple HTML Page

In this practical exercise, you'll create a simple HTML page to practice what you've learned so far. This exercise will help you understand the basic structure of an HTML document and how to use various HTML elements.

Steps to Create Your HTML Page:

  1. Open a text editor and create a new file named index.html.
  2. Start with the basic HTML structure: 
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First HTML Page</title>
</head>
<body>
    <h1>Hello World!</h1>
    <p>Welcome to my first HTML page.</p>
</body>
</html>
  3. Save your file and open it in a web browser to see your work.
