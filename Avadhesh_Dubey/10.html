Selectors, Properties, and Values

CSS selectors, properties, and values are fundamental concepts that determine how elements are styled on a webpage.

Selectors:

Selectors are patterns used to select the elements you want to style. Examples include:

  • Element Selector: p targets all <p> elements.
  • Class Selector: .classname targets all elements with the class classname.
  • ID Selector: #idname targets the element with the ID idname.
  • Attribute Selector: [type="text"] targets elements with a specific attribute.

Properties and Values:

Properties define what aspect of an element you are styling, and values specify the styles. Examples include:

  • Color Property: color: red; sets the text color to red.
  • Font Size Property: font-size: 16px; sets the font size to 16 pixels.
  • Margin Property: margin: 10px; adds a 10-pixel margin around the element.
